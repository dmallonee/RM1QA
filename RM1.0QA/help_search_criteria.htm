Create an Ad-Hoc or Scheduled Search

 

To begin, click on the Search Criteria tab (see highlighted area) at the top of the screen; then follow the instructions below:

 

Tab

Section

Action(s)

Quick Search (web only)

Profile

This section is to be used only for updating or saving profiles. Please skip this step until further notice.

 

City/Location

Select the pick up city – the return city will fill automatically

 

Rental Dates

Enter date range for search

 

 

Select absolute or relative dates

a.       If it is a one time search for the specific date range, you must select Date Absolute and the system will search only those dates

b.       If you want to save this search to your profile, you should select a relative date. This will allow you to search the profile a certain number of days out for the specified number of days

 

Days of the week

Select the days of the week you want the system to search; for example, if you want a weekday rate, you can select Tuesday or if you want to obtain weekday and weekend rates, you could select both Monday and Saturday

 

Details

Select  the Pick up and drop off time

 

 

Select the Length of Rent (1 – 32)

 

 

Select the Data Source

a.       Option 1: A specific travel site (Orbitz, Travelocity or Expedia)

b.       Option 2: All branded sites (Enterprise, Budget, Dollar, etc.) that are requested in the next section

 

Car types and companies

Highlight the car types and use the arrows to move the Car types you want included in your report into the selected column

 

 

Select the highlighted vendor and use the arrows to move the Car types you want included in your report into the selected column

a.       The highlighted vendor is the vendor in which the other vendor’s rates are compared to on reports. The highlighted vendor’s rates appear on the far left of all reports

 

 

Display Options

Please refer to your custom report, as its settings override this value

 

Rate Changes

Please refer to your custom report, as its settings override this value

 

Batch Shopping / Reporting Options

Presentation is pre-selected as html

 

 

Indicate whether or not you want an email to notify that a report is ready to view.

a.       Search – the report will be generated but no email notification will be sent when the report is ready to view

b.       Search and email all – the reports and alerts will be generated and an email notification will be sent when the rate report and or alert report is ready to view

c.       Search and email alerts – the alerts will be generated and an email notification will be sent when the alerts are ready to be viewed

Indicate in the Scheduled Action Time field if your search will be an ad-hoc search or the time you want your scheduled search to occur.

a.       For an ad-hoc search, select “Perform Search Now”  

b.       For a scheduled search, select the time that you want the report to begin searching. You will then need to check the day(s) of the week you want the search performed
